# 13: 创建群组（含跨群组导入）

**Status:** done
**Type:** implementation
**Blocked by:** 12

## What

用户从群组切换器或「我的」页面新建群组，可选是否从已有群组导入分类和菜品（复制语义）。

## Why

从零开始建群组门槛高——用户已有菜品池时，应能一键带过去。复制语义保证两份数据独立演进，不互扰。

## Acceptance

- [x] 新建群组页面：输入群组名 → 勾选"从已有群组导入"（可选）→ 选择源群组 → 勾选要导入的分类
- [x] 导入时：分类和该分类下的菜品、菜品图片 URL 全部复制到新群组，`creatorId` 保留原创建者 openid
- [x] 不导入抽取方案（新群组默认创建一套默认方案即可）
- [x] 不导入历史记录
- [x] 创建成功后自动切换到新群组
- [x] 群组名在当前用户的群组列表中不重名（仅需用户级唯一，非全局）
