# 05 — 菜品池管理

**What to build:** 在菜品池展示基础上添加完整的增删改操作：添加菜品、编辑菜品名称和图片、删除菜品、切换启用/禁用状态，以及图片上传至云存储。所有操作完成后菜品列表实时刷新。

**Blocked by:** 04 — 菜品池展示

**Status:** done

- [x] 菜品池页面提供「添加菜品」入口（浮动按钮或 tab 内按钮），默认归属当前所在分类
- [x] 添加/编辑表单：菜品名输入（最多 20 字，超出截断或提示），分类选择，图片上传区（最多 1 张）
- [x] 图片上传调用 `wx.chooseImage` + 云存储 `uploadFile`，返回 fileID 存入文档
- [x] 支持删除已上传图片（上传区可移除单张）
- [x] 编辑菜品：点击菜品卡片进入编辑表单，表单回填当前数据
- [x] 删除菜品：点击触发删除确认，确认后从 `dishes` 集合删除文档（关联云存储图片一并删除）
- [x] 启用/禁用切换：菜品卡片上提供操作开关，更新 `dishes.enabled` 字段，列表样式同步变化
- [x] 所有写操作完成后列表局部刷新（不全量重载）
