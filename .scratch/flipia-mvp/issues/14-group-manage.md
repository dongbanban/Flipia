## What

在「我的」页面提供当前群组管理入口。子页面包含：群组名编辑、成员列表、邀请、退出群组 / 解散群组。

## Why

用户需要管理群组成员、邀请家人加入、或不需要时离开/解散群组。权限模型：群主独占踢人/解散，其他操作所有成员平权。

## Acceptance

- [x] 群组管理页面入口：从「我的」页面进入
- [x] 群组名：点击可编辑，组内任意成员均可修改
- [x] 成员列表：展示头像 + 昵称列表（最多 5 人），群主可点击成员尾部 × 踢人（非群主无此按钮）
- [x] 邀请：点击"邀请成员"生成微信分享卡片（带 join code），被邀请人点击卡片后加入群组
- [x] 加入：被邀请人打开卡片 → 校验是否已满 5 人 → 校验是否已在群组内 → 加入成功 → 刷新群组列表
- [x] 退出：非群主成员点击"退出厨房" → 二次确认 → 从群组成员列表移除 → 从退出者的群组列表移除该群组
- [x] 解散：群主（创建者）在任何时候可主动解散厨房（需二次确认），销毁群组及所有数据。群主不显示退出按钮，非群主不显示解散按钮
- [x] 成员退出/被踢后，该成员加入的群组列表不再显示该厨房；厨房数据不受影响（留在厨房中）
