# Agent Skills

Engineering skills 的 per-repo 配置。

## Issue tracker

Issues 和 PRD 以本地 markdown 文件管理，存放在 `.scratch/` 目录下。详见 `docs/agents/issue-tracker.md`。

## Triage labels

使用默认的 5 角色标签词汇。详见 `docs/agents/triage-labels.md`。

## Domain docs

Single-context 布局。领域词汇在 `CONTEXT.md`，架构决策在 `docs/adr/`。详见 `docs/agents/domain.md`。
